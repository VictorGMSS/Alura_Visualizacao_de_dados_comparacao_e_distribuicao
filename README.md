# Alura_Visualizacao_de_dados_compara-o_distribui-o

# 📊 Data Visualization: Comparison & Distribution Charts

Projeto desenvolvido durante meus estudos de Visualização de Dados com foco em gráficos de comparação e distribuição utilizando Python e bibliotecas do ecossistema de Data Science.

---

## 🚀 Sobre o Projeto

Este projeto explora diferentes técnicas de visualização voltadas para comparação, distribuição e interpretação de dados, utilizando gráficos adequados para comunicar informações de forma clara, objetiva e visualmente eficiente.

Além da construção dos gráficos, também foram aplicadas práticas de design visual e storytelling com dados para destacar informações relevantes ao público.
